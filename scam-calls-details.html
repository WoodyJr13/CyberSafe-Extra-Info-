Scam/Fraud Calls: Recognizing Deceptive Phone Calls

  • Deceptive Calls: Attackers impersonate trusted entities (banks, government, tech support, family).
  • Goal: To coerce victims into revealing info, transferring money, or granting remote access.
  • Red Flags: Unsolicited calls, demands for immediate action, threats (e.g., arrest), requests for gift cards/wire transfers.
  • Prevention: Be skeptical, hang up, and call the organization back using an official number (not one provided by the caller).
